## Print specs

- **Trim**: A4, 210 x 297 mm. **Bleed**: 3 mm all edges, built into the file (page size 216 x 303 mm). Backgrounds extend to the page edge; all text sits at least 6 mm inside trim.
- **Color**: files are RGB (HTML/Chrome limitation). Ask the print shop to convert with their coated profile. Approximate CMYK for brand colors: red `#D63B1F` = 0/72/86/16; ink `#141210` = rich black (60/40/40/100) for the flyer-2 background, flat black for small text; paper `#F7F4EF` = 0/1/3/3.
- **Stock**: 250-300 gsm silk for handouts.
- **Re-export after edits**: open the HTML in Chrome, Print, Save as PDF, margins None, scale 100%, Background graphics ON. Verify the PDF is 216 x 303 mm. Confirm Bricolage Grotesque/Outfit loaded (needs internet) before exporting.
